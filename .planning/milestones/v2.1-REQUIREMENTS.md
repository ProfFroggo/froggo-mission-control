# Requirements: Writing System

**Defined:** 2026-02-12 (v2.0), updated 2026-02-13 (v2.1)
**Core Value:** Kevin can create a new book project by conversing with an AI agent that plans the story arc, chapter outline, themes, and characters -- then write in a 3-pane layout where AI chat dialogue drives content into the workspace.

## v2 Requirements (Complete)

All v2.0 requirements shipped 2026-02-13. 44/44 complete.

### Foundation
- [x] **FOUND-01**: User can create a new writing project with title and type
- [x] **FOUND-02**: User can see a list of all writing projects with stats
- [x] **FOUND-03**: User can open a writing project and see its chapter list
- [x] **FOUND-04**: User can create, rename, and delete chapters
- [x] **FOUND-05**: User can edit chapter content in a rich text editor
- [x] **FOUND-06**: Chapter content auto-saves with debounce
- [x] **FOUND-07**: User can navigate between chapters via outline sidebar
- [x] **FOUND-08**: Writing workspace accessible from main dashboard sidebar
- [x] **FOUND-09**: User can see word count per chapter and total project word count

### Editor
- [x] **EDIT-01**: Rich text editor supports headings, bold, italic, lists, blockquotes, links
- [x] **EDIT-02**: Editor loads from and saves to markdown files
- [x] **EDIT-03**: Editor handles 10k+ word chapters without performance degradation
- [x] **EDIT-04**: User can undo/redo changes with standard keyboard shortcuts

- [x] **FEED-04**: User can accept an alternative or dismiss
- [x] **FEED-05**: User can select which agent to send feedback to
- [x] **FEED-06**: AI response streams in real-time
- [x] **FEED-07**: Feedback interactions are logged per chapter
- [x] **FEED-08**: AI context includes current chapter, outline, and memory store data

- [x] **MEM-03**: User can create/edit/delete verified facts
- [x] **MEM-04**: Memory store displays in context panel
- [x] **MEM-05**: Memory store data automatically injected into AI context
- [x] **MEM-06**: Memory store persists as JSON files per project

- [x] **RES-02**: User can link sources to specific facts
- [x] **RES-03**: User can mark facts as verified/disputed/needs-source

## v2.1 Requirements (Complete)

### Chat Infrastructure
- [x] **CHAT-01**: User can chat with AI agents in a persistent chat pane alongside the editor
- [x] **CHAT-02**: AI responses stream in real-time (token by token) in the chat pane
- [x] **CHAT-03**: User can select which agent to chat with via agent picker
- [x] **CHAT-04**: Chat history persists across sessions
- [x] **CHAT-05**: AI context includes current chapter content, project outline, and memory store data
- [x] **CHAT-06**: User can insert AI-generated content from chat into the editor with one click
- [x] **CHAT-07**: Content insertion validates against TipTap schema via markdown conversion
- [x] **CHAT-08**: User can copy and retry chat messages
- [x] **CHAT-09**: Chat maintains multi-turn conversation context via gateway sessions
- [x] **CHAT-10**: Chat pane uses separate gateway sessions from inline feedback

### Three-Pane Layout
- [x] **LAYOUT-01**: Writing workspace uses 3-pane layout
- [x] **LAYOUT-02**: Panes are resizable via drag handles
- [x] **LAYOUT-03**: Chat pane and chapters sidebar are individually collapsible
- [x] **LAYOUT-04**: Pane sizes and collapse state persist across sessions
- [x] **LAYOUT-05**: Layout works at window widths from 1024px to 1920px+

### Setup Wizard
- [x] **WIZARD-01**: User can start a new book project through a conversational AI wizard
- [x] **WIZARD-02**: User can provide a brain dump / free-form description as starting point
- [x] **WIZARD-03**: Wizard generates character profiles and populates memory store
- [x] **WIZARD-04**: Wizard generates a chapter outline
- [x] **WIZARD-05**: Wizard generates story arc / plot structure summary
- [x] **WIZARD-06**: User can review and edit the proposed plan before creating
- [x] **WIZARD-07**: User can skip the wizard and use quick-create
- [x] **WIZARD-08**: Wizard state persists across navigation and app restarts
- [x] **WIZARD-09**: Wizard uses agent-specialized prompts
- [x] **WIZARD-10**: On completion, wizard atomically creates project with all data populated
- [x] **WIZARD-11**: User can select or let AI infer the genre/type beyond memoir/novel
