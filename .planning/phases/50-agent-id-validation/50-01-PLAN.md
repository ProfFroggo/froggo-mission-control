  <files>src/lib/validateId.ts</files>
  <action>
Create `src/lib/validateId.ts` exporting two things:

1. `AGENT_ID_PATTERN = /^[a-z0-9][a-z0-9-_]*$/` — lowercase alphanumeric, hyphens, underscores; must start with alphanumeric (prevents leading `-` or `_` edge cases). Max length 64 chars.

2. `validateAgentId(id: unknown): NextResponse | null` — returns a `NextResponse.json({ error: 'Invalid agent ID' }, { status: 400 })` if the id is not a non-empty string matching AGENT_ID_PATTERN and length <= 64. Returns `null` if valid (null = passes, caller continues). Import NextResponse from 'next/server'.

Keep it minimal — no classes, no throws. Pattern: if invalid → return error response, if valid → return null.

Example usage at call site:
```ts
const guard = validateAgentId(id);
if (guard) return guard;
// safe to use id below
```
