- Fields: name (text input), relationship (text input), description (textarea), traits (text input, comma-separated)
- useState for each field, pre-fill from character prop if editing
- On submit: parse traits by splitting on commas and trimming. Call addCharacter or updateCharacter from memoryStore. Call onCancel to close form.
- Cancel button calls onCancel
- Simple layout: stacked fields with labels, small text inputs matching clawd design (same as ChapterSidebar input styling)
