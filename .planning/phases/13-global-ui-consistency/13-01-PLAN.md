<task type="auto">
  <name>Task 1: Add missing CSS custom properties to index.css</name>
  <files>src/index.css</files>
  <action>
In `src/index.css`, add three new CSS custom properties to the `:root` block (dark theme defaults) and the `:root.light` block (light theme overrides).

Add to the `:root` block, AFTER the existing `--clawd-accent-dim` line and BEFORE `--clawd-font`:

```css
--clawd-bg-alt: #1a1a1a;   /* Alternate bg: between bg (#0a0a0a) and surface (#141414), used for input backgrounds */
--clawd-bg0: #0a0a0a;      /* Deepest background, same as --clawd-bg, used for status dots and inactive states */
--clawd-card: #141414;     /* Card background, same as --clawd-surface, used for message bubbles and cards */
```

Add to the `:root.light` block, AFTER the existing `--clawd-text-dim` line:

```css
--clawd-bg-alt: #f4f4f5;   /* Light alternate bg: slightly darker than white for input backgrounds */
--clawd-bg0: #fafafa;      /* Deepest background in light mode */
--clawd-card: #ffffff;     /* Card background in light mode */
```

These values are chosen to maintain the existing visual hierarchy:
- `bg0` = deepest (matches `bg`) for inactive/status indicators
- `bg-alt` = between bg and surface, for input fields inside surface-colored containers
- `card` = matches `surface`, for elevated card/bubble backgrounds
  </action>
