**3. Enhanced `parseAlternatives` to extract commentary:**

Change the return type and logic of `parseAlternatives` to support an optional `commentary` field. The function should accept an optional `agentId` parameter:

```typescript
interface ParsedAlternative {
  text: string;
  commentary?: string;
}

function parseAlternatives(response: string, agentId?: string): ParsedAlternative[] {
  const parts = response.split(/###\s*Alternative\s*\d+\s*/i);
  return parts.slice(1).map(p => {
    if (agentId === 'jess') {
      const whyMatch = p.match(/\*\*Why:\*\*\s*([\s\S]*?)$/i);
      if (whyMatch) {
        const text = p.slice(0, whyMatch.index).trim();
        const commentary = whyMatch[1].trim();
        return { text, commentary };
      }
    }
    return { text: p.trim() };
  }).filter(a => a.text).slice(0, 3);
}
```
