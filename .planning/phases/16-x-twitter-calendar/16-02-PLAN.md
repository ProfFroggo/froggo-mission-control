**B. XCalendarView.tsx -- Add isEventDraggable and onCreateTweet**

Add to the XCalendarView component:

```typescript
function isEventDraggable(event: CalendarEvent): boolean {
  return event.colorId === 'scheduled';
}
```

2. A Create Tweet handler that dispatches a custom DOM event to switch tabs:
```typescript
const handleCreateTweet = useCallback(() => {
  window.dispatchEvent(new CustomEvent('x-tab-change', { detail: 'drafts' }));

```

```typescript
<EpicCalendar
  externalEvents={events}

```

**C. XTwitterPage.tsx -- Listen for x-tab-change custom event**

In XTwitterPage, add a useEffect that listens for the custom event:
```typescript
import { useState, useEffect } from 'react'; // Add useEffect to import

// Inside XTwitterPage, after the useState:
useEffect(() => {
  const handler = (e: Event) => {
    const tab = (e as CustomEvent).detail as XTab;
    if (tab) setActiveTab(tab);
  };
  window.addEventListener('x-tab-change', handler);
  return () => window.removeEventListener('x-tab-change', handler);
}, []);
```

This wires the "Create Tweet" button on the calendar to navigate to the drafts tab.
