# Plan 03-01: Tasks API Routes

**Phase**: 3 — API Routes
**Goal**: All task-domain IPC handlers converted to `/api/tasks/*` REST routes

## Tasks

### Task 1: Create tasks collection route
`app/api/tasks/route.ts` — GET with status/assignedTo/project/priority filters, JSON field parsing, priority sort. POST with generated id.

### Task 2: Create task detail routes
`app/api/tasks/[id]/route.ts` — GET, PATCH (dynamic SET), DELETE.

### Task 3: Subtask routes
`app/api/tasks/[id]/subtasks/route.ts` — GET + POST with auto-position.
`app/api/tasks/[id]/subtasks/[subtaskId]/route.ts` — PATCH + DELETE.

### Task 4: Activity + Attachments routes
`app/api/tasks/[id]/activity/route.ts` — GET + POST.
`app/api/tasks/[id]/attachments/route.ts` — GET + POST.
