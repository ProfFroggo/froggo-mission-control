    **3. Create `electron/writing-version-service.ts`:**

    Following the exact pattern of `writing-research-service.ts`:

    - Import: `ipcMain` from `electron`, `fs`, `path`, `writingProjectPath, writingChapterPath, writingVersionsPath` from `./paths`, `diffWords, Change` from `diff`.

    - Define `VersionMeta` interface: `{ id: string; chapterId: string; label: string; createdAt: string; filename: string; wordCount: number; }`.

    - Helper `readVersionsManifest(projectId, chapterId)`: Read `versions.json` from `writingVersionsPath(projectId, chapterId)`, return `VersionMeta[]`. Return `[]` if file doesn't exist.

    - Helper `writeVersionsManifest(projectId, chapterId, versions)`: Write JSON array to `versions.json`.

    - Helper `stripHtml(html: string): string`: Strip HTML tags for clean prose diffs. Replace `<[^>]+>` with empty, decode `&nbsp;` `&amp;` `&lt;` `&gt;`.

    - `saveSnapshot(projectId, chapterId, label?)`:
      1. Read `chapters.json` from project dir to find chapter by ID.
      2. Read chapter content from `writingChapterPath(projectId, chapter.filename)`.
      3. Generate `id = "ver-" + Date.now() + "-" + random4chars`, `filename = "v-" + Date.now() + ".md"`.
      4. Create version dir with `fs.promises.mkdir(writingVersionsPath(projectId, chapterId), { recursive: true })`.
      5. Write content to `path.join(versionsDir, filename)`.
      6. Read existing manifest, push new `VersionMeta`, write back.
      7. Return `{ success: true, version: meta }`.

    - `listVersions(projectId, chapterId)`: Read manifest, return sorted by createdAt descending.

    - `readVersionContent(projectId, chapterId, versionId)`: Find version in manifest, read file content, return `{ success: true, content }`.

    - `restoreVersion(projectId, chapterId, versionId)`:
      1. Read version content from snapshot file.
      2. Find chapter in chapters.json.
      3. Write version content to `writingChapterPath(projectId, chapter.filename)`, overwriting current.
      4. Return `{ success: true }`.

    - `computeDiff(projectId, chapterId, versionId)`:
      1. Read version content (snapshot file).
      2. Read current chapter content.
      3. `stripHtml()` both.
      4. `const changes = diffWords(strippedVersion, strippedCurrent)`.
      5. Return `{ success: true, changes, versionLabel: version.label }`.

    - `deleteVersion(projectId, chapterId, versionId)`:
      1. Find version in manifest, remove entry, delete file, write updated manifest.
      2. Return `{ success: true }`.

    - `export function registerWritingVersionHandlers()`:
      Register 6 IPC handlers:
      - `writing:version:list` -> listVersions
      - `writing:version:save` -> saveSnapshot
      - `writing:version:read` -> readVersionContent
      - `writing:version:restore` -> restoreVersion
      - `writing:version:diff` -> computeDiff
      - `writing:version:delete` -> deleteVersion
      Each handler should wrap in try/catch returning `{ success: false, error: message }` on failure.
      Log: `console.log('[writing-version] IPC handlers registered');`

    **1. Create `src/store/versionStore.ts`:**

    Following the pattern of `researchStore.ts`:

    ```typescript
    import { create } from 'zustand';

    interface VersionMeta {
      id: string;
      chapterId: string;
      label: string;
      createdAt: string;
      filename: string;
      wordCount: number;
    }

    interface DiffChange {
      value: string;
      added?: boolean;
      removed?: boolean;
    }

    interface VersionState {
      versions: VersionMeta[];
      loading: boolean;
      activeDiff: { changes: DiffChange[]; versionLabel: string } | null;
      diffLoading: boolean;

      loadVersions: (projectId: string, chapterId: string) => Promise<void>;
      saveVersion: (projectId: string, chapterId: string, label?: string) => Promise<boolean>;
      deleteVersion: (projectId: string, chapterId: string, versionId: string) => Promise<void>;
      restoreVersion: (projectId: string, chapterId: string, versionId: string) => Promise<boolean>;
      loadDiff: (projectId: string, chapterId: string, versionId: string) => Promise<void>;
      clearDiff: () => void;
      reset: () => void;
    }
    ```

    Use `const bridge = () => (window as any).clawdbot?.writing?.version;` accessor pattern.

    Actions:
    - `loadVersions`: Set loading, call `bridge().list(projectId, chapterId)`, set `versions` from result, clear loading.
    - `saveVersion`: Call `bridge().save(projectId, chapterId, label)`. If success, re-call `loadVersions` to refresh list. Return success boolean.
    - `deleteVersion`: Call `bridge().delete(...)`, then re-call `loadVersions`.
    - `restoreVersion`: Call `bridge().restore(...)`. Return success boolean. (Caller must reload chapter content.)
    - `loadDiff`: Set diffLoading, call `bridge().diff(projectId, chapterId, versionId)`, set `activeDiff` from result `{ changes, versionLabel }`, clear diffLoading.
    - `clearDiff`: Set `activeDiff` to null.
    - `reset`: Clear versions, activeDiff, loading states.
