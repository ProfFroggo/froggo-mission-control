    Modify ChapterSidebar to wrap the chapter list in DnD context and add a collapsible toggle, following the pattern from `EditPanelsModal.tsx`:

    1. Import DnD: `DndContext, closestCenter, PointerSensor, KeyboardSensor, useSensor, useSensors, DragEndEvent` from `@dnd-kit/core`; `SortableContext, verticalListSortingStrategy, sortableKeyboardCoordinates, arrayMove` from `@dnd-kit/sortable`.

    2. Import `ChevronDown` from `lucide-react` (for collapsible toggle).

    3. Add state: `const [chaptersCollapsed, setChaptersCollapsed] = useState(false);`

    4. Set up sensors (exact pattern from EditPanelsModal):
       ```
       const sensors = useSensors(
         useSensor(PointerSensor, { activationConstraint: { distance: 5 } }),
         useSensor(KeyboardSensor, { coordinateGetter: sortableKeyboardCoordinates }),
       );
       ```

    5. Add `reorderChapters` and `openProject` from useWritingStore destructure (openProject is already available through activeProject, but we need the action to refresh after reorder). Actually, add `reorderChapters` to the existing destructure. After reorder, call `openProject(activeProject.id)` to refresh chapter list from disk.

    6. Add handleDragEnd:
       ```
       const handleDragEnd = async (event: DragEndEvent) => {
         const { active, over } = event;
         if (over && active.id !== over.id) {
           const oldIndex = chapters.findIndex(c => c.id === active.id);
           const newIndex = chapters.findIndex(c => c.id === over.id);
           const newOrder = arrayMove(chapters, oldIndex, newIndex);
           await reorderChapters(newOrder.map(c => c.id));
         }
       };
       ```

    7. Add a "Chapters" section header with collapsible toggle between the "Add chapter" section and the chapter list. It should show "Chapters ({count})" with a ChevronDown icon that rotates when collapsed. Style: `px-3 py-1.5 flex items-center justify-between text-[10px] uppercase tracking-wider text-clawd-text-dim`. The ChevronDown rotates 180deg via `transform rotate-180` when NOT collapsed (pointing down = expanded, pointing right = collapsed). Use a `transition-transform` for smooth rotation.

    8. When `chaptersCollapsed` is true, hide the chapter list (don't render the list items). When false, wrap the existing chapter list map in:
       ```
       <DndContext sensors={sensors} collisionDetection={closestCenter} onDragEnd={handleDragEnd}>
         <SortableContext items={chapters.map(c => c.id)} strategy={verticalListSortingStrategy}>
           {chapters.map(...)}
         </SortableContext>
       </DndContext>
       ```

    9. The empty state ("No chapters yet") should NOT be wrapped in DnD context.
  </action>
