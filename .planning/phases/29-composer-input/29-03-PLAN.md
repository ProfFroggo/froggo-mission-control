Read `SessionStatsBar.tsx` first to understand what token data it receives/displays. Then add a subtle context ring to the composer.

**Context Ring design:**
- A small circular arc indicator (SVG, ~20px) positioned in the bottom-left of the composer
- Shows % of context window used (current tokens / model max)
- Color: green < 50%, yellow 50-80%, red > 80%
- On hover: shows a tooltip with "X / Y tokens (Z%)"

**Step 1: Add a ContextRing component** (add near the top of ChatPanel.tsx, after imports):

```tsx
function ContextRing({ used, max }: { used: number; max: number }) {
  if (!used || !max) return null;
  const pct = Math.min(used / max, 1);
  const r = 8; const circ = 2 * Math.PI * r;
  const color = pct > 0.8 ? '#ef4444' : pct > 0.5 ? '#f59e0b' : '#22c55e';
  return (
    <div className="relative group" title={`${used.toLocaleString()} / ${max.toLocaleString()} tokens (${Math.round(pct * 100)}%)`}>
      <svg width="20" height="20" viewBox="0 0 20 20" className="rotate-[-90deg]">
        <circle cx="10" cy="10" r={r} fill="none" stroke="currentColor" strokeWidth="2" className="text-mission-control-border" />
        <circle
          cx="10" cy="10" r={r} fill="none"
          stroke={color} strokeWidth="2"
          strokeDasharray={circ}
          strokeDashoffset={circ * (1 - pct)}
          strokeLinecap="round"
        />
      </svg>
    </div>
  );
}
```

When the user types @ in the ChatRoomView composer, show a dropdown of agents in the room. Selecting an agent inserts a pill token (visible, styled span) into the composer. Pills are sent as @agentId mentions in the message.

**Step 1: Add mention state**

In ChatRoomView, add state for mentions:
```tsx
const [mentionQuery, setMentionQuery] = useState<string | null>(null); // null = no mention active
const [mentionedAgents, setMentionedAgents] = useState<string[]>([]); // selected agent IDs
```

**Step 2: Detect @ in input change handler**

In the `onChange` handler of the composer textarea, detect when the user types @:
```tsx
onChange={(e) => {
  const val = e.target.value;
  setInput(val);
  // Detect @mention trigger
  const atMatch = val.match(/@(\w*)$/);
  if (atMatch) {
    setMentionQuery(atMatch[1].toLowerCase());
  } else {
    setMentionQuery(null);
  }
}}
```

**Step 3: Agent mention dropdown**

When `mentionQuery !== null`, show a dropdown of filtered room agents:
```tsx
{mentionQuery !== null && room && (() => {
  const roomAgentIds: string[] = (() => {
    try { return JSON.parse(room.agents || '[]'); } catch { return []; }
  })();
  const filtered = agents.filter(a =>
    roomAgentIds.includes(a.id) &&
    (a.name?.toLowerCase().includes(mentionQuery) || a.id.includes(mentionQuery))
  );
  if (filtered.length === 0) return null;
  return (
    <div className="absolute bottom-full left-0 mb-1 bg-mission-control-surface border border-mission-control-border rounded-lg shadow-lg z-50 min-w-48 max-h-48 overflow-y-auto">
      {filtered.map(agent => (
        <button key={agent.id}
          className="w-full px-3 py-2 text-left text-sm hover:bg-mission-control-bg flex items-center gap-2"
          onMouseDown={(e) => {
            e.preventDefault(); // prevent blur
            // Replace @query with @agentId
            const newInput = input.replace(/@\w*$/, '');
            setInput(newInput);
            setMentionedAgents(prev => [...new Set([...prev, agent.id])]);
            setMentionQuery(null);
          }}>
          <span className="font-medium">{agent.name}</span>
          <span className="text-mission-control-text-dim text-xs">@{agent.id}</span>
        </button>
      ))}
    </div>
  );
})()}
```
