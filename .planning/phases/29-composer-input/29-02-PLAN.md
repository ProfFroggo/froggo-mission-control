Add localStorage-based draft persistence that saves the input text per agent and restores it when the agent is selected.

**Step 1: Add draft save/load helpers** (near top of the component or as module-level functions):

```tsx
const DRAFT_PREFIX = 'froggo:chat-draft:';

function saveDraft(agentId: string, text: string) {
  try {
    if (text.trim()) {
      localStorage.setItem(DRAFT_PREFIX + agentId, text);
    } else {
      localStorage.removeItem(DRAFT_PREFIX + agentId);
    }
  } catch { /* localStorage unavailable in SSR */ }
}

function loadDraft(agentId: string): string {
  try {
    return localStorage.getItem(DRAFT_PREFIX + agentId) || '';
  } catch { return ''; }
}
```

**Step 2: Load draft when agent changes**

Find the `useEffect` or callback that runs when `selectedAgent` changes. After switching agents, load their draft:

```tsx
useEffect(() => {
  if (selectedAgent?.id) {
    const draft = loadDraft(selectedAgent.id);
    setInputValue(draft); // or whatever the input state setter is named
  }
}, [selectedAgent?.id]);
```

**Step 3: Save draft on every keystroke**

Find the onChange handler for the input textarea. After updating `inputValue`, also save the draft:

```tsx
onChange={(e) => {
  const val = e.target.value;
  setInputValue(val);
  if (selectedAgent?.id) saveDraft(selectedAgent.id, val);
}}
```

**Step 4: Clear draft when message is sent**

In `handleSend` (or wherever the message send logic is), after successfully sending, clear the draft:
```tsx
if (selectedAgent?.id) saveDraft(selectedAgent.id, '');
```
