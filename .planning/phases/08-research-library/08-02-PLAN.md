1. Create `src/store/researchStore.ts` following the exact pattern from `memoryStore.ts`:
   - Types: `ResearchSource` (id, title, author, type: 'book'|'article'|'interview'|'website'|'document'|'other', url, notes, created_at, updated_at) and `FactSourceLink` (fact_id, source_id, notes, created_at).
   - State: `sources: ResearchSource[]`, `factSourceMap: Record<string, string[]>` (factId -> sourceId[]), `loading: boolean`, `editingId: string | null`.
   - `bridge()` accessor: `(window as any).clawdbot?.writing?.research`.
   - Actions:
     - `loadSources(projectId)` — calls bridge().sources.list(), sets sources array.
     - `clearSources()` — resets sources, factSourceMap, editingId.
     - `setEditingId(id)`.
     - `addSource(projectId, data)` — calls bridge().sources.create() then re-lists.
     - `updateSource(projectId, id, data)` — calls bridge().sources.update() then re-lists.
     - `deleteSource(projectId, id)` — calls bridge().sources.delete() then re-lists. Clear editingId if matches.
     - `loadLinksForFact(projectId, factId)` — calls bridge().links.forFact(), updates factSourceMap[factId].
     - `linkSourceToFact(projectId, factId, sourceId)` — calls bridge().links.link() then reloads links for that fact.
     - `unlinkSourceFromFact(projectId, factId, sourceId)` — calls bridge().links.unlink() then reloads links.
     - `loadAllFactLinks(projectId, factIds: string[])` — for each factId, loadLinksForFact. Used on tab switch.

     ```typescript
     function buildFactCheckPrompt(
       claim: string,
       chapterContext: string,
       sources: { title: string; author: string; type: string }[],
       facts: { claim: string; source: string; status: string }[],
     ): string {
       const sourcesList = sources.length > 0
         ? sources.map(s => `- "${s.title}" by ${s.author} (${s.type})`).join('\n')
         : '(no sources in library yet)';
       return [
         'You are a meticulous research editor focused on accuracy and fact-checking.',
         '',
         '## Task',
         'The user has highlighted a claim and wants you to fact-check it.',
         '',
         '### Claim to Verify',
         `"${claim}"`,
         '',
         '### Chapter Context',
         chapterContext || '(no context available)',
         '',
         '### Research Library (existing sources)',
         sourcesList,
         '',
         '### Known Facts',
         facts.length > 0
           ? facts.map(f => `- [${f.status}] ${f.claim}`).join('\n')
           : '(no facts recorded yet)',
         '',
         '## Response Format',
         'Respond with:',
         '1. **Verdict:** VERIFIED | DISPUTED | NEEDS MORE RESEARCH',
         '2. **Confidence:** HIGH | MEDIUM | LOW',
         '3. **Explanation:** Brief explanation of your finding',
         '4. **Suggested Sources:** If you can identify specific sources that would help verify this claim',
         '5. **Suggested Status:** What status this fact should have (verified/disputed/needs-source)',
       ].join('\n');
     }
     ```
