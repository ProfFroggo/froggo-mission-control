**The correct consolidated definition** for `component-patterns.css`:
```css

  background: var(--color-panel);
  border: 1px solid var(--gray-a4);

.glass-card:hover {
  border-color: var(--gray-a6);
  box-shadow: 0 4px 16px var(--black-a5);
}
```
Using `var(--color-panel)` (Radix panel bg) instead of `var(--gray-a2)` for better theme alignment.
