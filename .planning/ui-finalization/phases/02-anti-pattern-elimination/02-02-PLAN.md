# Plan 02-02: Remove Gradient Text

<objective>
Remove all `bg-clip-text text-transparent` gradient text across the codebase. This is a banned
pattern — solid token colors only. Eight instances found: 5 in DashboardRedesigned.tsx, 1 in
Dashboard.tsx, 1 in NewContentWidget.tsx, 1 in InboxWidget.tsx.

For KPI stat numbers (the 5xl metric values), replace the gradient with the semantic color
directly (e.g., `text-[var(--color-danger)]`). For headings/titles, use `text-mission-control-text`.
</objective>
