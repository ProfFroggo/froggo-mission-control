## Decision: Semantic green taxonomy
Green is the semantic color for **approve / accept / allow / join** actions throughout Mission Control.
All other primary action CTAs use the default accent color (violet).

**Keep green:**
- ClaraReviewDashboard "Approve and dispatch agent"

- XDraftListView "Bulk Approve" drafts
- XAgentContentQueue "Approve and Schedule" / "Batch Approve"
- MeetingsPanel all approve/accept action items
- NotificationsPanelV2 approval button
- AgentDetailModal "Allow" permission override

**Convert to violet (remove color prop):**
- EnhancedSettingsPanel 7× `color="grass"` Save buttons
- AccessibilitySettings "Test announcement" button
- MarketplaceBrowse "Install" button
- WorkerModal 3× `color="grass"` create/submit buttons
- XDraftListView "New Draft" compose button (×2)
