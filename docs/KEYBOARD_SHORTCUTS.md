## Tips & Tricks

### Discoverability
- **Hover** over buttons to see tooltips with shortcuts
- Look for **keyboard hint badges** on buttons (e.g., `N`, `⌘S`)
- Press **?** in any panel for panel-specific shortcuts

### Context-Aware
- `⌘N` adapts to your current panel:
  - In **Tasks** → New Task
  - In **X** → New Tweet
  - In **Inbox** → depends on context
- `⌘F` opens filters in Tasks, search in other panels

### Safety
- Shortcuts won't trigger while you're typing in text fields
- Destructive actions (like delete) ask for confirmation
- You can always press `Esc` to cancel

### Efficiency
- Use `⌘1-9` for instant navigation (no mouse needed!)
- Combine with panel shortcuts: `⌘5` (Tasks) → `N` (New Task)
- Master the Inbox shortcuts (J/K/A/R/X) for rapid approval

## Platform Notes

**macOS:**
- `⌘` = Command key
- All shortcuts use ⌘ as primary modifier

**Windows/Linux:**
- `Ctrl` key replaces ⌘
- All shortcuts work the same way
