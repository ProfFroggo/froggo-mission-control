## 💡 Tips

1. **Tab Switching**: Use `⌘1-5` in any panel with tabs to quickly jump between sections
2. **Context Aware**: Some shortcuts (like `⌘N`) do different things depending on context
3. **Escape Key**: Always closes modals/panels, even when typing in inputs
4. **Cmd vs Ctrl**: Use `Cmd` on macOS, `Ctrl` on Windows/Linux
5. **Visual Hints**: Look for keyboard shortcut hints in button tooltips
