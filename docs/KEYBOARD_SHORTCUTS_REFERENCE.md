- Type naturally: `"fix login bug"`
- Filter by type: `task: login bug`
- Filter by status: `status:in-progress`
- Filter by agent: `agent:coder`
- Filter by priority: `priority:p0`

## Shortcut Conflicts

### Avoiding Conflicts

**If shortcut doesn't work:**
1. Check if macOS is using it (System Preferences > Keyboard > Shortcuts)
2. Check if browser extension is using it
3. Check if another app has captured it

**Common conflicts:**
- `⌘⇧Space` - Spotlight/Siri (macOS)
- `⌘Space` - Spotlight (macOS)
- `⌘Tab` - App switcher (macOS)
- `⌘,` - Settings (standard macOS)

**Resolution:**
- Disable macOS shortcut if not needed
- Or: Customize Froggo shortcut to different keys

## Tips for Mastery

**1. Start Small**
- Learn 3-5 shortcuts per week
- Use them until muscle memory
- Then add more

**2. Print This Reference**
- Keep near desk
- Refer when needed
- Highlight your favorites

**3. Disable Mouse**
- Challenge: 1 day keyboard-only
- Forces learning
- Dramatically improves speed

**4. Customize**
- Settings > Keyboard > Custom
- Map frequent actions to easy shortcuts
- Make it yours

**5. Teach Others**
- Best way to remember
- Share tips with team
- Build keyboard-first culture
