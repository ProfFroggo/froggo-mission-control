## Platform-Specific Notes

- **⌘** = Command (macOS) / Ctrl (Windows/Linux)
- **⇧** = Shift
- **⌫** = Delete/Backspace
- **Esc** = Escape

## Design Principles

1. **Consistency**: Similar actions use similar shortcuts across panels
2. **Mnemonics**: Shortcuts match action names (N=New, S=Save, R=Refresh)
3. **Progressive**: ⌘ alone for basic, ⌘⇧ for advanced actions
4. **Context-aware**: Shortcuts adapt to the active panel
5. **Non-blocking**: Single keys (J/K/N) only in list contexts
